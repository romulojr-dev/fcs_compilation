\textbf{Given:}
\[
J_1 = 4~\mathrm{kg\,m^2},\; J_2 = 7~\mathrm{kg\,m^2},\; J_3 = 5~\mathrm{kg\,m^2}
\]
\[
K_1 = 12~\mathrm{N\,m/rad},\; K_2 = 15~\mathrm{N\,m/rad}
\]
\[
B_1 = 4~\mathrm{N\,m\,s/rad},\; B_2 = 6~\mathrm{N\,m\,s/rad},\; B_3 = 4~\mathrm{N\,m\,s/rad}
\]
\[
N_1 = 8,\; N_2 = 12,\; N_g = 10
\]
\[
T \; \text{(applied to bottom inertia).}
\]

\textbf{Required:}
\[
H(s) = \frac{\theta_3(s)}{T_i(s)}
\]

\textbf{Solution:}

Reflected inertias to shaft of \( J_3 \):
\[
J_1' = J_1 N_g^2 = 4 \cdot 10^2 = 400
\]
\[
J_2' = J_2 \left( \frac{N_g}{N_1} \right)^2 = 7 \cdot \left( \frac{10}{8} \right)^2 = 7 \cdot \frac{100}{64} = \frac{700}{64}
\]
\[
J_3 = 5
\]

Total equivalent inertia:
\[
J_{\text{eq}} = J_1' + J_2' + J_3 = 400 + \frac{700}{64} + 5 = \frac{26620}{64}
\]

Reflected spring constants:
\[
K_1' = K_1 N_g^2 = 12 \cdot 100 = 1200
\]
\[
K_2 = 15
\]

Total equivalent stiffness:
\[
K_{\text{eq}} = K_1' + K_2 = 1200 + 15 = 1215
\]

Reflected damping:
\[
D_1' = D_1 N_g^2 = 4 \cdot 100 = 400
\]
\[
D_2' = D_2 \left( \frac{N_g}{N_1} \right)^2 = 6 \cdot \frac{100}{64} = \frac{600}{64}
\]
\[
D_3 = 4
\]

Total equivalent damping:
\[
D_{\text{eq}} = D_1' + D_2' + D_3 = 400 + \frac{600}{64} + 4 = \frac{26660}{64}
\]

Equation of motion:
\[
T(s) = J_{\text{eq}} s^2 \theta_3(s) + D_{\text{eq}} s \theta_3(s) + K_{\text{eq}} \theta_3(s)
\]

Transfer function:
\[
\frac{\theta_3(s)}{T(s)} = \frac{1}{J_{\text{eq}} s^2 + D_{\text{eq}} s + K_{\text{eq}}}
\]

Substitute values:
\[
\frac{\theta_3(s)}{T(s)} = \frac{1}{\frac{26620}{64} s^2 + \frac{26660}{64} s + 1215}
\]

Multiply numerator and denominator by 64:
\[
\frac{\theta_3(s)}{T(s)} = \frac{64}{26620 s^2 + 26660 s + 77760}
\]

\[
\boxed{\frac{\theta_3(s)}{T(s)} = \frac{64}{26620 s^2 + 26660 s + 77760}}
\]

\textbf{Given:}
\[
J_1 = 3.6~\mathrm{kg\,m^2},\;\; J_2 = 6.8~\mathrm{kg\,m^2},\;\; J_3 = 5.4~\mathrm{kg\,m^2}
\]
\[
K_1 = 14~\mathrm{N\,m/rad},\;\; K_2 = 18~\mathrm{N\,m/rad}
\]
\[
B_1 = 3.5~\mathrm{N\,m\,s/rad},\;\; B_2 = 5.5~\mathrm{N\,m\,s/rad},\;\; B_3 = 4.2~\mathrm{N\,m\,s/rad}
\]
\[
N_1 = 9,\;\; N_2 = 13,\;\; N_g = 11
\]

\textbf{Required:}
\[
H(s) \equiv \dfrac{\Theta_3(s)}{T(s)}
\]

\textbf{Solution:}

Define the kinematic (magnitude) gear ratios referenced to shaft 1:
\[
r_{12}=\frac{N_1}{N_2}=\frac{9}{13},\qquad
r_{13}=\frac{N_1}{N_g}=\frac{9}{11}.
\]

Reflect inertias/dampers/springs to a single reference shaft (shaft 1).
\[
\begin{aligned}
J_{\rm eq} &= J_1 + r_{12}^2 J_2 + r_{13}^2 J_3,\\[4pt]
B_{\rm eq} &= B_1 + r_{12}^2 B_2 + r_{13}^2 B_3,\\[4pt]
K_{\rm eq} &= K_1 + r_{13}^2 K_2.
\end{aligned}
\]

Torque reflection
\[
\tau_{\rm eq} = -\,r_{13}\,T,
\]

Single-DOF equation on shaft 1.
\[
J_{\rm eq}\,\ddot\theta_1 + B_{\rm eq}\,\dot\theta_1 + K_{\rm eq}\,\theta_1 = -\,r_{13}\,T(t).
\]

Taking Laplace transforms (zero initial conditions) and solving for \(\Theta_1(s)/T(s)\):
\[
\Theta_1(s)\big( J_{\rm eq}s^2 + B_{\rm eq}s + K_{\rm eq}\big)
= -\,r_{13}\,T(s)
\]
\[
\frac{\Theta_1(s)}{T(s)} = \frac{-\,r_{13}}{J_{\rm eq}s^2 + B_{\rm eq}s + K_{\rm eq}}.
\]

Using \(\Theta_3(s) = -\,r_{13}\,\Theta_1(s)\) we get
\[
\frac{\Theta_3(s)}{T(s)} \;=\; \frac{r_{13}^2}{J_{\rm eq}s^2 + B_{\rm eq}s + K_{\rm eq}}
\]

First the squared ratios (decimal):
\[
r_{12}^2=\left(\frac{9}{13}\right)^2=\frac{81}{169}\approx 0.4792899408,
\]
\[
r_{13}^2=\left(\frac{9}{11}\right)^2=\frac{81}{121}\approx 0.6694214876.
\]

Now compute the equivalent parameters (showing arithmetic):
\[
J_{\rm eq} = 3.6 + \left(\frac{9}{13}\right)^2 (6.8) + \left(\frac{9}{11}\right)^2 (5.4)
\]
\[
= 3.6 + 0.4792899408 \times 6.8 + 0.6694214876 \times 5.4
\]
\[
= 3.6 + 3.2591715974 + 3.6148760333
\]
\[
\approx 10.4740476307\ \mathrm{kg\cdot m^2}
\]

\[
B_{\rm eq} = 3.5 + \left(\frac{9}{13}\right)^2 (5.5) + \left(\frac{9}{11}\right)^2 (4.2)
\]
\[
= 3.5 + 0.4792899408 \times 5.5 + 0.6694214876 \times 4.2
\]
\[
= 3.5 + 2.6360946744 + 2.8115732420
\]
\[
\approx 8.9476679164\ \mathrm{N\cdot m\cdot s/rad}
\]


\[
K_{\rm eq} = 14 + \left(\frac{9}{11}\right)^2 (18)
\]
\[
= 14 + 0.6694214876 \times 18
\]
\[
= 14 + 12.0495867769
\]
\[
\approx 26.0495867769\ \mathrm{N\cdot m/rad}
\]


Using \(r_{13}^2\approx 0.6694215\), \(J_{\rm eq}\approx 10.4740\), \(B_{\rm eq}\approx 8.9477\), \(K_{\rm eq}\approx 26.0496\):

\[
\boxed{\;
H(s) \;=\; \frac{\Theta_3(s)}{T(s)} \;=\;
\frac{0.6694215}{\,10.4740\,s^2 + 8.9477\,s + 26.0496\,}
\;}
\]

\textbf{Given:}
\[
J_1 = 6~\mathrm{kg\cdot m^2},\;\; J_2 = 4~\mathrm{kg\cdot m^2}
\]
\[
K_1 = 15~\mathrm{N\cdot m/rad},\;\; K_2 = 9~\mathrm{N\cdot m/rad},\;\; K_3 = 8~\mathrm{N\cdot m/rad}
\]
\[
B_1 = 4~\mathrm{N\cdot m\cdot s/rad},\;\; B_2 = 3.5~\mathrm{N\cdot m\cdot s/rad},\;\; B_3 = 2.5~\mathrm{N\cdot m\cdot s/rad}
\]
\[
N_1 = 10,\;\; N_2 = 7
\]
\[
T \text{ (applied to the inertia } J_1\text{).}
\]

\vspace{4pt}
\textbf{Required:}
\[
H(s) = \frac{\theta_2(s)}{T(s)}.
\]

\vspace{4pt}
\textbf{Solution:}

Reflect inertias through the gear ratios to combine them at the input:
\[
J_{\rm eq} = J_1 + \left(\frac{N_1}{N_2}\right)^2 J_2
\]
\[
J_{\rm eq} = 6 + \left(\frac{10}{7}\right)^2 (4)
\]
\[
J_{\rm eq} = 6 + 2.0408163265 \times 4
\]
\[
J_{\rm eq} = 6 + 8.163265306
\]
\[
J_{\rm eq} \approx 14.163~\mathrm{kg\cdot m^2}
\]

Reflect dampers through the gear ratios:
\[
B_{\rm eq} = B_1 + \left(\frac{N_1}{N_2}\right)^2 B_2 + B_3
\]
\[
B_{\rm eq} = 4 + \left(\frac{10}{7}\right)^2 (3.5) + 2.5
\]
\[
B_{\rm eq} = 4 + 7.142857143 + 2.5
\]
\[
B_{\rm eq} \approx 13.643~\mathrm{N\cdot m\cdot s/rad}
\]

Reflect springs through the gear ratios:
\[
K_{\rm eq} = K_1 + \left(\frac{N_1}{N_2}\right)^2 K_2 + K_3
\]
\[
K_{\rm eq} = 15 + \left(\frac{10}{7}\right)^2 (9) + 8
\]
\[
K_{\rm eq} = 15 + 18.36734694 + 8
\]
\[
K_{\rm eq} \approx 41.367~\mathrm{N\cdot m/rad}
\]

Write the equation of motion for the equivalent single inertia system:
\[
J_{\rm eq} \ddot{\theta} + B_{\rm eq} \dot{\theta} + K_{\rm eq} \theta = T(s)
\]
\[
(J_{\rm eq} s^2 + B_{\rm eq} s + K_{\rm eq}) \Theta(s) = T(s)
\]

\[
H(s) = \frac{\Theta(s)}{T(s)} = \frac{1}{J_{\rm eq} s^2 + B_{\rm eq} s + K_{\rm eq}}
\]
\[
H(s) = \frac{1}{14.163 s^2 + 13.643 s + 41.367}
\]

\[
\boxed{H(s) \approx \frac{1}{14.163 s^2 + 13.643 s + 41.367}}
\]
